# Contributing to awesome-free-models

Thanks for your interest in contributing!

## How to Contribute

1. **Fork the repository**
2. **Add your model/resource** in the appropriate section
3. **Follow the format** (see below)
4. **Submit a Pull Request**

## Format Guidelines

### Adding a Model

```markdown
- [ModelName](https://huggingface.co/org/model) - Brief description. [[License]](link)
```

### Adding a Tool or Provider

```markdown
- [ToolName](https://example.com/) - Brief description. [GitHub](https://github.com/user/repo)
```

### Requirements for Inclusion

- Must be genuinely free to use (not "free trial with credit card required")
- Must be a real, working project or service
- Include the model's license when possible
- Prefer links to Hugging Face for models, GitHub for tools

### Sections

- **Open-Weight Models** - Models you can download and run yourself
- **Free API Providers** - Services offering free API access
- **Local Inference Tools** - Software to run models locally
- **Embedding Models** - Models for semantic search and RAG
- **Multimodal & Vision Models** - Image/video understanding models
- **Audio & Speech Models** - Speech-to-text, TTS, music generation

## Quality Standards

- Links should be permanent (no redirect chains)
- Prefer permissively licensed models (Apache 2.0, MIT, CC-BY)
- Clearly note any usage restrictions or license limitations
- Avoid dead repos and broken links

## Thank You!

Your contributions help the community discover free AI resources.
